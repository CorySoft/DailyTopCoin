DailyTopCoinHelper
============

Description:

    淘金币助手 - 帮助淘宝用户实现批量账户的淘金币自动自动领取。

    DailyTopCoinHelper
        Helping user@taobao.com get daily TopCoins automatically.

Requires:
    requests, BeautifulSoup

Usage:
    python DailyTopCoinHelper.py
    
使用方法：
首先编辑accounts.txt,按照下面的格式保存账号密码:

    {
        "username": "password",
        "username": "password",
        "username": "password",
        "username": "password",
        "username": "password"
    }
    
需要多少个就添加多少个。

初版代码简陋，只是实现了基本功能，代码会逐渐慢慢重构。
